== Description ==

Mage FAQ is one of the best free faq plugin in WordPress. You can use it for Troubleshooting, FAQ, Q&A etc

Features:

* Shortcode Powred
* Unlimited Category
* Unlimited FAQ Question
* Open Source
* Developer Friendly
* Use Shortcode [mkb] for showing all faq question. if you want to show last 5 faq question use [mkb show=5], if you want to show faq of a particular category use [mkb cat=CatID] if you want to show last 5 faq question of particular category use [mkb cat=CatID show=5].
* Live demo http://demo.wp-people.net/mage-faq/

== Installation ==

Upload the Mage FAQ plugin to your blog, Activate it, then use shortcode as your need).

1, 2, 3: You're done!
